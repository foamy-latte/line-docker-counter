### Prerequisites

- Clean installation of Ubuntu host
- Internet connection

### Installation

From within the line-docker-counter directory:
```bash
chmod +x install.sh
sudo install.sh
```

### Coverage/tests

```bash
sudo ./test_no_coverage.sh
```
